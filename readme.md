# CSV Plotter #
This is a tool that plots csv files from our datalogger.

## Installation ##
### Python ###
Download [this](https://www.python.org/downloads/).

Open a command prompt. On windows, go to the search bar and type 'cmd', and command prompt should pop up first.

Type `python --version` to check that it's installed and added to your PATH. If it isn't, try rebooting your computer.

Type `pip install matplotlib`. This installs the plotter used by csv plotter. If pip is not found, you can also try `python -m pip install matplotlib`.

Type `pip install dearpygui`. This installs the gui library for csv plotter. If pip is not found, you can also try `python -m pip install dearpygui`.

### CSV Plotter ###

You can either clone the repository or just download the file `plot.py`.

## Running ##

Once python and matplotlib are installed you should just be able to click on the file plot.py and run it.
